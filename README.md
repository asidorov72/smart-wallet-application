# Getting Started

### Configure mysql

Configure mysql on following parameters:

> host: `localhost`\
> port: `3307`\
> user: `root`\
> password: `1234`\
> database: `smart_wallet_app_may_2026`

### Run mysql in the docker container

For working with MySQL database do not forget to run:

> docker start mysql-standalone

### Open project

Open the project in browser:

> http://localhost:8093/
